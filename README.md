# ❤모여라 동글❤
### 👉 Casual Puzle Game
> 동글들을 선으로 이어서 가장 높은 점수를 얻는 안드로이드 모바일 게임입니다.

## 👉 프로젝트 소개
- 게임 장르 : 물리기반 2D 퍼즐
- 플랫폼 : 안드로이드 모바일
- 제작 인원 : 1 인 (개인 프로젝트, 디자인과 개발 모두 직접 제작)
- 제작 기간 : 14 일 (3/2 ~ 3/16)
- 사용 엔진과 언어 : Unity, C#
